# Huffman Encoder & Decoder

This repository contains an encoder & decoder for files using the [Huffman](https://en.wikipedia.org/wiki/Huffman_coding) method.

## To Run
- Use the included makefile
- Run ``./huffman``
- See ``./huffman -h`` for help
